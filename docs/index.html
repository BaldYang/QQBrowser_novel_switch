欢迎使用「QQ浏览器小说文本转换器」

该软件基于iApp制作，用于提取转换QQ浏览器中已下载的小说文本，QQ浏览器中下载的小说文本为网页文件，直接阅读体验不佳，该软件将此文本进行分段、截取，转换出可阅读的txt文本，并进行合并，可用于导入部分阅读器查看或直接查看。
（推荐Android11以下系统使用）
